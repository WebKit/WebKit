Test for Bugzilla Bug 86885: CSS2.1 failure: border-conflict-element021a.

When two adjacent table row groups (thead, tbody, tfoot) have the same border-width and the same border-style in a 'border-collapse: collapse' table, then the color of the border from the topmost table row group wins: so, a thead wins over a tbody which wins over tfoot.

The bottom border of the first (green) box should be completely green in color (including the corners) and overlapping the top border of the second (orange) box. Similarly, the bottom border of the second (orange) box should be completely orange in color (including the corners) and overlapping the top border of the third (green) box.
