Prerequisites
Browsers that are subjected to this test should support CSS1 and the CSS2 table model, and multiple classes.

If this paragraph breaks the line here: and here: then your browser does support table values on 'display'. If this paragraphs flows without any undue breaks, then you should mark your browser as not supporting table values!
