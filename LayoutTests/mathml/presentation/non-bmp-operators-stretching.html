This test verifies the stretching of some non-BMP operators. The font used by this test provides horizontal stretching data for U+1EEF0, U+1EEF1 and U+1D400. It is assumed that U+1EEF0 and U+1EEF1 are horizontal and only have a prefix form with property stretchy. It is also assumed that U+1D400 is not in the operator dictionary. Hence only the prefix operators U+1EEF0 and U+1EEF1 will stretch horizontally.

U+1EEF0:𞻰 U+1EEF1:𞻱 U+1D400:𝐀

Infix form:

U+1EEF0:𞻰 U+1EEF1:𞻱 U+1D400:𝐀

Postfix form:

U+1EEF0:𞻰 U+1EEF1:𞻱 U+1D400:𝐀
