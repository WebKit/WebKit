This test verifies the spacing of some non-BMP operators. It is assumed that U+1EEF0 and U+1EEF1 only have a prefix form with zero lspace/rspace and that U+1D400 is not in the operator dictionary. Hence the operator spacing use zero for prefix U+1EEF0/U+1EEF1 and will use the default thickmathspace value otherwise. Note that for implicit form, WebKit has a workaround for bad markup and will fallback to the form of a dictionary entry if one is found.

U+1EEF0:𞻰_ U+1EEF1:𞻱_ U+1D400:𝐀_

Postfix:

U+1EEF0:𞻰_ U+1EEF1:𞻱_ U+1D400:𝐀_

Infix:

U+1EEF0:𞻰_ U+1EEF1:𞻱_ U+1D400:𝐀_
