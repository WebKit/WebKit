This test verifies that a selection in a region is painted correctly when a user drags it.

Steps:

  1. Starting at "Some text", select all of the text in the yellow box.
  2. While the text is selected, press the mouse down on the link. Do not release the mouse.
  3. Drag the selection to the whitespace on the right.

Expected Results:

  • A faded image of the selected text and link should follow the mouse as you drag.
  • The faded image should only contain the selected text and link.
  • The faded image should have a transparent background, not yellow and not gray.
