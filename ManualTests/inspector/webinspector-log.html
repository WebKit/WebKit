Test for Bug 28047: add utility method to enable logging messages for inspector development.

  • Open a browser, go to any web page.
  • Open web inspector
  • Open the console panel
  • Open a web inspector on the web inspector
  • Open that inspector's console panel
  • In that panel, run the code: 
       WebInspector.log("hello, world")

Results:
In the original inspector window, there should now be a message in the console with the text "hello, world". There will be a blue maginifying glass icon to the left of the message. Run the same command again, and you should see the repeat count go to 2. The message and repeat count are blue.
