Test left to right popup list box's width is restricted within the left screen.

  1. Setup dual screens.
  2. Launch browser and maximize the browser window size in the left screen.
  3. Enter a very long string into "Left to right" text input box.
  4. Hit enter.
  5. Go back and start to enter the same string again.
  6. Make sure the popup list box's width which suggests the result doesn't exceed the left screen width.

Test right to left popup window width is restricted within the right screen.

  1. Setup dual screens.
  2. Launch browser and maximize the browser window size in the right screen.
  3. Enter a very long string into "Right to Left" text input box.
  4. Hit enter.
  5. Go back and start to enter the same string again.
  6. Make sure the popup list box's width which suggests the result doesn't exceed the right screen width.
