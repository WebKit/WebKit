HTML Test Index

The HTML tests are available in several variants.

Tests With Navigation Aids

Each category of test is available using several different harnesses. The name of the harness describes how the test markup is contained within it, for example the Xlink embed case uses an XLink with the show axis set to embed.

  • full: Self Contained, XHTML <iframe>, XHTML <object>, XHTML <frame>, HTML <iframe>, HTML <object>, HTML <frame>, XLink embed, TNG Format
  • static: Self Contained, XHTML <iframe>, XHTML <object>, XHTML <frame>, HTML <iframe>, HTML <object>, HTML <frame>, XLink embed, TNG Format
  • history-related: Self Contained, XHTML <iframe>, XHTML <object>, XHTML <frame>, HTML <iframe>, HTML <object>, HTML <frame>, XLink embed, TNG Format
  • interactive: Self Contained, XHTML <iframe>, XHTML <object>, XHTML <frame>, HTML <iframe>, HTML <object>, HTML <frame>, XLink embed, TNG Format
  • dynamic: Self Contained, XHTML <iframe>, XHTML <object>, XHTML <frame>, HTML <iframe>, HTML <object>, HTML <frame>, XLink embed, TNG Format
